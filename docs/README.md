## Digital Minds — Reading Summaries (Indicator Gaming)

This folder contains **8 short summaries** of the most relevant papers/posts for our project: **experimental demonstrations of indicator gaming for AI consciousness / AI moral patiency (moral status) indicators**.

### What’s in here
- **Start here**: `docs/index.md`
- **Summaries**: `docs/01-...md` through `docs/08-...md`
- **Template for new summaries**: `docs/_template.md`

### How to keep this up to date
- **ArXiv papers**: check if there’s a newer version (v2, v3, …) and update the “Last reviewed” line.
- **Lab blog posts / system cards** (Anthropic/OpenAI/Apollo): revisit quarterly; these pages change over time.
- **When adding a new resource**:
  - copy `docs/_template.md`
  - keep it short (aim: 5–10 min read)
  - add it to `docs/index.md`
